1.The concat() Method

DEFINTION : The concat() method concatenates (joins) two or more arrays.The concat() method does not change the existing arrays

SYNTAX: array1.concat(array2, array3, ..., arrayX)

2.The slice() Method

DEFINTION : The slice() method selects from a given start, up to a (not inclusive) given end.The slice() method does not change the original array.

SYNTAX : array.slice(start, end)

3.The splice() Method

DEFINTION : The splice() method adds and/or removes array elements.The splice() method overwrites the original array.

SYNTAX : array.splice(index, howmany, item1, ....., itemX)

4.The reverse() Method

DEFINTION : The reverse() method reverses the order of the elements in an array.The reverse() method overwrites the original array.

SYNTAX : array.reverse()

5.The sort() Method

DEFINTION : The sort() sorts the elements of an array.The sort() sorts the elements as strings in alphabetical and ascending order.

SYNTAX : array.sort(compareFunction)

6.The some() Method

DEFINTION : The some() method checks if any array elements pass a test (provided as a callback function).The some() method returns true (and stops) if the function returns true for one of the array elements.The some() method returns false if the function returns false for all of the array elements.The some() method does not execute the function for empty array elements.

SYNTAX : array.some(function(value, index, arr), this)
